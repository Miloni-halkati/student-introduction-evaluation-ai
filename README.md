# AI-Powered Student Introduction Scoring Tool

This project evaluates a student's self-introduction transcript using:
- Salutation scoring
- Keyword presence scoring
- Flow/structure scoring
- Words per minute (WPM)
- Grammar score (LanguageTool)
- Vocabulary richness (TTR)
- Filler word rate
- Sentiment positivity score

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

3. Paste transcript or upload file to get full scoring.
